# AZM Store Bot — دليل التثبيت البسيط

---

## الخطوة 1 — افتح ملف bot.py وعدّل هذين السطرين فقط

افتح ملف `bot.py` بأي محرر نصوص وابحث في أعلى الملف عن:

```python
BOT_TOKEN = "8635803760:..."      ← توكن البوت (موجود بالفعل)
HOST_DOMAIN = ""                   ← ضع هنا دومين سيرفرك
```

**مثال بعد التعديل:**
```python
HOST_DOMAIN = "mysite.com"
```

> إذا كان السيرفر بـ IP مباشر بدون دومين:
> ```python
> HOST_DOMAIN = "1.2.3.4:8081"
> ```

---

## الخطوة 2 — نصّب المكتبات (مرة واحدة فقط)

```bash
pip install -r requirements.txt
```

---

## الخطوة 3 — شغّل البوت

```bash
python main.py
```

هذا الأمر يشغّل **الخادم والبوت معاً** في نفس الوقت.

---

## ملاحظات مهمة

| | |
|---|---|
| قاعدة البيانات | `products.db` — لا تحذفها |
| صور المنتجات | مجلد `uploads/` |
| Mini App | `webapp/dist/public/` — جاهزة، لا تحتاج بناء |

---

## إذا كان عندك Nginx أو بانل (cPanel/DirectAdmin)

اجعل الدومين يحوّل الطلبات إلى البورت `8081`، ثم ضع الدومين في `HOST_DOMAIN`.

---

## استكشاف الأخطاء

| المشكلة | الحل |
|---------|------|
| زر "فتح المتجر" لا يعمل | تأكد أن `HOST_DOMAIN` مكتوب صح في `bot.py` |
| صفحة بيضاء عند فتح المتجر | تأكد أن `webapp/dist/public/index.html` موجود |
| البوت لا يرد | تأكد أن `BOT_TOKEN` صحيح |
